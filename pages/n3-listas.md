1. **Listas NO ordenadas**
<br> Se crean con asteriscos (*), guiones (-) o signos de más (+), seguido de un espacio al inicio de cada elemento de la lista.
<br> Usando sangrías, se pueden crear sublistas anidadas.
<br>![Lista no ordenada](/assets/listaNoOrdenada.png)

2. **Listas ordenadas**
<br> Se crean con números seguidos de un punto (.) y un espacio al inicio de cada elemento de la lista. Ni siquiera es necesario que los números estén en orden, el resultado siempre será una lista ordenada.
<br> Usando sangrías, se pueden crear sublistas anidadas.
<br>![Lista ordenada](/assets/listaOrdenada.png)


2. **Listas mixtas**
<br> Se pueden combinar listas ordenadas y no ordenadas para crear listas mixtas.
<br> Usando sangrías, se pueden crear sublistas anidadas.
<br>![Lista mixta](/assets/listaMixta.png)
